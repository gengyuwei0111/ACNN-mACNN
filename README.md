## Training Usage
To train the PRNN for a problem on given domain and draw a graph for regression
```shell
python ./train_model.py
 --case 2 
 --dim 2 
 --hidden_layers 2 
 --hidden_neurons 20 
 --lam_adf 1 
 --lam_bd 1 
 --optimizer 'Adam' 
 --Test_Mode 'LocalFitting' 
 --epochs_Adam 5000 
 --epochs_LBFGS 200 
 --TrainNum 2000 
 --coeff_para 50 
 --sigma 0.01 
 --domain 0 1
```

## Testing  Usage
To evaluate numerical error and relative sensitivity
```shell
python ./evaluate_model.py
 --case 2 
 --dim 2 
 --hidden_layers 2 
 --hidden_neurons 20 
 --lam_adf 1 
 --lam_bd 1 
 --optimizer 'Adam' 
 --Test_Mode 'LocalFitting'
```
